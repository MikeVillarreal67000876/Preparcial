# Pokemon API - Pre Parcial

API desarrollada con FastAPI para gestionar pokemones de la primera generación.

## Tecnologías usadas
- Python
- FastAPI
- Pydantic
- Uvicorn
- CSV
- Git y GitHub

## Endpoints
- `/showallpokemons/`
- `/showonepokemon/`
- `/showonepokemonbyid/`
- `/pokemonbattle/`
- `/pokemonorderedby/`

## Ejecutar el proyecto
```bash
uvicorn main:app --reload
